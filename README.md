ACGNNAV - 综合性次元航标站

ACGNNAV 是一站式次元文化与技术探索导航站。本项目以 ACGN 内容为核心，延伸至 AI 生产力、云基础设施及各类互联网实用工具，旨在为爱好者和开发者提供一个纯净、高效的链接直达中心。


---

🌐 在线访问

👉 https://acgnnav.top


---

🌐 站点核心 (ACGN Core)

本站深度覆盖二次元全产业链内容：

Animation (动画)：精品番剧、剧场版及在线观看/下载资源分享。

Comic (漫画)：海内外优质漫画平台与汉化资源直达。

Game (游戏)：单机/端游/手游资讯、MOD 社区及下载地址。

Novel (小说)：轻小说、网络文学分类导航及阅读工具推荐。


---

🛠️ 技术与生产力 (Tech & Productivity)

除了二次元内容，我们还关注前沿技术应用：

AI 工具箱：收录主流大模型、AI 绘图、自动化助手及效率工具。

基础设施：精选全球优质云服务器 (VPS) 厂商、域名注册商及网络优化方案。

实用工具：网盘资源、国内外优秀软件下载、临时邮箱、IP 查询及网络测速等实用链接。


---

🚀 技术架构

本项目基于成熟的开源方案搭建，具有极高的稳定性与响应速度：

核心引擎：WordPress

导航主题：WebStack WordPress 版 (MIT License)

服务器环境：Ubuntu 22.04 LTS + LNMP (Nginx 1.28 + MySQL 5.7 + PHP 8.2)

安全防护：全站 SSL 加密 (Let's Encrypt) 访问。


---

📖 安装说明

如果你想搭建类似的导航站，可以参考以下步骤：

准备一台 KVM 架构的 VPS（推荐 Ubuntu 系统）。

部署宝塔面板并安装 LNMP 环境。

创建 WordPress 站点并配置 Nginx 伪静态规则。

下载并启用 WebStack 主题，刷新固定链接。
